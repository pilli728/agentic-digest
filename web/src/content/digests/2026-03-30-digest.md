## What Shipped

*Anthropic's worst-kept secret is now everyone's problem.*

### [Claude Mythos (Capybara) Leaked via Anthropic's Own CMS](https://fortune.com/2026/03/26/anthropic-says-testing-mythos-powerful-new-ai-model-after-data-leak-reveals-its-existence-step-change-in-capabilities/)
A config error in Anthropic's content management system exposed ~3,000 unpublished assets, including draft docs about their next model: Claude Mythos, codenamed Capybara. It sits above Opus as a new tier. The leaked docs say it gets "dramatically higher scores" on coding, reasoning, and cybersecurity benchmarks. The real story: Anthropic's own safety docs warn it's "far ahead of any other AI model in cyber capabilities" and could let attackers outpace defenders. They're giving cyber defense teams early access before opening it up.

### [ARC-AGI-3: Humans 100%, Best AI 0.37%](https://arcprize.org/arc-agi/3)
New interactive benchmark where AI agents have to explore game environments, figure out the objective on their own, and execute. No instructions, no hints. Humans solve every single one. Gemini 3.1 Pro scored 0.37%. GPT-5.4 got 0.26%. Opus 4.6 managed 0.25%. Grok-4.20 literally scored zero. There's $700K on the table for the first agent to hit 100%. We're clearly not at AGI yet, and this benchmark is going to be the scoreboard that proves it.

## What's Moving

*Ship faster or get shipped.*

### [Stripe's Machine Payments Protocol: Agents Can Now Pay for Things](https://stripe.com/blog/machine-payments-protocol)
Stripe launched MPP on March 18, an open protocol for AI agents to make payments without any human in the loop. Agents can pay in stablecoins or fiat, handle micropayments, and spin up sessions on services like Browserbase. Visa is already onboard. This is part of Stripe's broader Agentic Commerce Suite, which also includes an MCP server with 25 tools, docs search, and integrations for Cursor, Vercel AI SDK, LangChain, and more. If you build anything with agents and money, this is now table stakes.

### [Clone Any Website in One Prompt with Claude Code](https://github.com/ericshang98/perfect-web-clone)
Perfect Web Clone is a multi-agent system built on Claude Agent SDK with 40+ tools. The key insight: it reads the actual DOM, CSS, and structured blocks instead of screenshotting the page and guessing. You point it at a URL and it extracts design tokens, assets, and component structure, then dispatches parallel builder agents to reconstruct everything. The output is production-ready code, not a rough approximation. Also worth checking out: [JCodesMore's cloner template](https://github.com/JCodesMore/ai-website-cloner-template) which does similar via Chrome MCP.

### [Remotion + Claude Code = Videos from Prompts](https://www.remotion.dev/docs/ai/claude-code)
Remotion (React for video) launched Agent Skills in January, and the combo with Claude Code is now the default way devs make product videos, data visualizations, and animated explainers. You describe what you want in plain English. Claude writes the React/TypeScript components. Remotion renders an MP4. The package hit 25,000 installs in days. This isn't replacing After Effects for motion designers, but for dev content and product demos it's absurdly good.

### [Sierra Ghostwriter: An Agent That Builds Agents](https://x.com/btaylor/status/2036858449032863898)
Bret Taylor's Sierra released Ghostwriter this week. You describe your customer experience in plain English (or feed it SOPs, transcripts, even photos of whiteboard sketches) and it builds a production-ready agent that handles chat, phone, email, and 30+ languages. The interesting part: it runs a continuous improvement loop, analyzing real customer interactions, spotting where it underperforms, testing fixes in a sandbox, and deploying updates automatically. Clients include Nordstrom, Rivian, and SiriusXM.

### [30 Days of Social Content in 2 Hours with Claude](https://www.threads.com/@thepromptvault_/post/DTj0SWggg3m/)
A viral thread broke down how to use Claude to plan, design, edit, and schedule a full month of social media content. The trick: set up your CLAUDE.md with brand voice, content pillars, and platform constraints once. Then use [Blotato](https://www.mindstudio.ai/blog/blotato-claude-code-social-media-automation-2) to publish across 9 platforms. Combined with Claude Code's new `/loop` feature (basically a cron scheduler in plain English), you can have it running content pipelines in the background. Not a replacement for genuine creativity, but a huge time saver for the repetitive stuff.
